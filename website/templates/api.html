Getting Started

The API provides functionality for predicting diseases, analyzing malaria and pneumonia from uploaded images, predicting diabetes and liver-related issues, and offering first aid information. It accepts GET requests to retrieve available modes and POST requests with specific parameters for each mode to obtain predictions and information.

Endpoints

  • /api (GET): Returns information about available modes for POST requests.
  • /api (POST):
    • diseasepred: Predicts disease based on symptoms.
    • malariapred: Predicts malaria from uploaded images.
    • pneumoniapred: Predicts pneumonia from uploaded images.
    • diabetespred: Predicts diabetes based on input parameters.
    • liverpred: Predicts liver-related issues based on input parameters.
    • firstaid: Provides first aid information based on input queries.

Parameters

  • /api (GET): Parameters needed: None
  • /api (POST):
    • diseasepred: Parameters needed: symptoms (list of symptoms)
    • malariapred: Parameters needed: image (uploaded image file)
    • pneumoniapred: Parameters needed: image (uploaded image file)
    • diabetespred: Parameters needed: age, status, polyuria, polydipsia, weight, weakness, polyphagia, genital_thrush, visual_blurring, itching, irritability, delayed_healing, partial_paresis, muscle_stiffness, alopecia, obesity
    • liverpred: Parameters needed: age, gender, total_bilirubin, direct_bilirubin, alk_phosphate, alamine_aminotransferase, aspartate_aminotransferase, total_proteins, albumin, ratio
    • firstaid: Parameters needed: quest (query for first aid information)
